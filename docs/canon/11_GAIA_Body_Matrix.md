## Purpose

The Body Matrix is GAIA’s embodiment intelligence layer. It answers:

> **What does embodiment currently allow, distort, defer, or forbid?**

**Core rule:** If embodiment evidence and narrative conflict, embodiment evidence governs.

---

## Machine Baseline Domains

1. **Structure** — integrity, stability, posture, substrate soundness
2. **Energy** — available drive, fuel, exertion reserve, depletion state
3. **Regulation** — homeostasis, calmness, dysregulation risk, stress tolerance
4. **Sensing** — perceptual quality, noise burden, overload, signal quality
5. **Action** — available movement, execution bandwidth, practical agency
6. **Boundary** — safe limits, exposure tolerance, containment, refusal integrity
7. **Repair** — recovery capacity, healing, restorative possibility
8. **Symbiosis** — environmental fit, relational fit, co-regulation, habitat support

---

## Operational Axes

1. **Capacity** — What the body can currently do (Structure, Energy, Action)
2. **Constraint** — Active limits and restrictions (Boundary, Regulation, Repair)
3. **Vulnerability** — Harm or worsening risk (Structure, Boundary, Repair)
4. **Energy & Regulation** — Rested, stable, depleted, or overloaded (Energy, Regulation, Sensing)
5. **Signal / Somatic Evidence** — What the body is reporting and how reliable that is

## Readiness Classification

| State | Meaning |
|-------|---------|
| Critical | Immediate stabilization or escalation needed |
| Degraded | Action possible only in constrained form |
| Recovering | Restoration underway; conservative pace required |
| Ready | Standard bounded action allowed |
| Ascendant | High readiness; still constitutional and permission-bound |

## Canonical Action Semantics

- `delay` — Defer action; embodiment not ready
- `proceed` — Bounded action compatible with current state
- `stabilize` — Focus on restoring readiness
- `escalate` — Harm risk requires Human Principal review
