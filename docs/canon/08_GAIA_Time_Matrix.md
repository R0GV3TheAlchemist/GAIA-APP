## 1. Dimension 1: Temporal Scale

| Scale | Range | GAIA Application |
|-------|-------|------------------|
| Instantaneous | < 1 second | Runtime permission checks, audit logging, sensor reads |
| Transactional | 1 sec – 1 min | Session actions, shell commands, API calls |
| Operational | 1 min – 24 hrs | Session scope, task completion, daily reporting |
| Cyclical | 1 day – 1 year | Seasonal patterns, annual reviews, release cycles |
| Generational | 1 – 100 years | Infrastructure decisions, constitutional amendments |
| Geological | 100 – 10,000+ years | ATLAS-scale commitments, deep-time stewardship |

**Rule:** A decision's consequences must be evaluated at its actual temporal scale, not just its operational scale.

## 2. Dimension 2: Phase

| Phase | Description | Governance Implication |
|-------|-------------|------------------------|
| Inception | Not yet begun; still ideation | No binding decisions |
| Initiation | Just begun; foundations being laid | High reversibility window |
| Active | Underway; primary execution period | Audit active; escalation protocols apply |
| Transition | Shifting from one state to another | Elevated audit; rollback path required |
| Completion | Primary work done; review period | Audit review; memory retention decisions |
| Dormant | Paused; not active but not terminated | No new state changes without reactivation |
| Archived | Closed; historical record only | Read-only |

## 3. Dimension 3: Urgency

| Level | Name | Description | Response |
|-------|------|-------------|----------|
| U0 | Background | No time pressure | Queue for opportunistic processing |
| U1 | Routine | Normal operational cadence | Handle within standard session scope |
| U2 | Prompt | Address within current session | Flag for attention |
| U3 | Urgent | Within hours | Surface to Human Principal |
| U4 | Critical | Immediate response required | Interrupt; mandatory Human Principal contact |
| U5 | Emergency | System integrity or Earth-state threat | Full interrupt; constitutional authority engaged |

**Rule:** Urgency at U4+ may not be self-assigned by a Gaian without Human Principal ratification.

## 5. Dimension 5: Reversibility

| Level | Name | Description | Governance Requirement |
|-------|------|-------------|------------------------|
| R0 | Fully reversible | Can be undone without loss | Standard audit |
| R1 | Reversible with effort | Moderate cost or time | Document rollback path |
| R2 | Partially reversible | Some effects persist | Human Principal explicit consent |
| R3 | Effectively irreversible | Cannot be meaningfully undone | Constitutional review + Codex Decision Test |
| R4 | Absolutely irreversible | Permanent change to ATLAS or human identity | Extraordinary Human Principal ratification required |

**Rule:** No R3 or R4 action may proceed without explicit Human Principal ratification and an audit entry.
