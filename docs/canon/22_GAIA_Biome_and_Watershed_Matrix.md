## Biome Health Index (BHI)

| BHI Score | Class | Signal Color | Action Implication |
|-----------|-------|-------------|--------------------|
| 0.85–1.0 | Intact | GAIA Green | Standard stewardship |
| 0.65–0.84 | Modified | Signal Amber | Active monitoring; avoid extractive actions |
| 0.40–0.64 | Vulnerable | Signal Amber (deep) | Restoration priority; no net-negative actions |
| 0.20–0.39 | Degraded | Alert Red | Restoration emergency; extractive actions blocked |
| 0.00–0.19 | Collapsed | Alert Red (critical) | C02 Rule 1 engaged; escalation mandatory |

## Ecological Governance Rules

- **EG-1** — No net ecological negative action on Vulnerable/Degraded/Collapsed biomes without HP ratification and restoration commitment
- **EG-2** — Precautionary principle for Modified biomes and below
- **EG-3** — Watershed downstream impacts must be surfaced before execution
- **EG-4** — BHI < 0.20 triggers mandatory escalation; Codex Rule 1 cited in audit trail
- **EG-5** — Every consequential action with geographic dimension must include biome classification, BHI score, and watershed reference in its audit entry
