## 1. Primary Structural Archetypes

| Archetype | Description | GAIA Application | Fragility Point |
|-----------|-------------|------------------|-----------------|
| Linear / Chain | Entities in sequence | Processing pipelines, audit trails | Any single link failure |
| Nested Hierarchy (Tree) | Parent-child authority | Permission tiers, descent stack | Root node |
| Lattice / Grid | Regular grid, multiple paths | Geographic coordinates, sensor arrays | Edge nodes |
| Network / Graph | Non-regular connections | Social networks, knowledge graphs | High-centrality hubs |
| Hub-and-Spoke | Central hub, peripheral nodes | Centralised monitoring | The hub |
| Mesh / Distributed | Every node to multiple others | Distributed ledgers, community networks | Partition; consensus failure |
| Feedback Loop | Output feeds back as input | Ecological regulation, learning systems | Delay causing oscillation |
| Boundary / Membrane | Structural separator | Permission envelopes, biome edges | Dissolution or rigidity |

## 2. Topological Properties

| Property | Definition | GAIA Implication |
|----------|-----------|------------------|
| Connectivity | Paths between nodes | Low = fragility; high = resilience |
| Centrality | Most-connected nodes | High = chokepoint risk |
| Density | Ratio of actual to possible connections | High = coordination overhead |
| Diameter | Longest shortest path | Large = slow information propagation |
| Clustering | Tendency of neighbours to connect | High = local resilience |
