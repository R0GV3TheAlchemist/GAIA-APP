## Superlattice Performance Equation

The stacked GAIANITE piezoelectric coefficient is governed by:

```
d₃₃(stack) = f(p, r, θ) · d₃₃(unit)
```

Where:
- **p** = periodicity (number of stacked units)
- **r** = thickness ratio between sublayer types
- **θ** = rotational orientation between stack units
- **d₃₃(unit)** = baseline Tier 1 coefficient (~12 pC/N)

Optimizing (p, r, θ) is the primary experimental target for Tier 2 synthesis. Literature confirms p = 3 is sufficient for initial demonstration of enhancement. [1]
